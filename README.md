# SIO-Quest

Un jeu développé sur les alternants de la promotion SIO 2026 du BTS SIO.

## Technologies utilisées

### Godot Engine
Godot est un moteur de jeu open-source très polyvalent, particulièrement performant pour la création de jeux en 2D. Nous l'utilisons pour programmer la logique du jeu, gérer les collisions et structurer les différentes scènes de notre projet.

### Pixel Studio
Pixel Studio (disponible sur Steam) est une application dédiée à la création de pixel art. Cet outil nous sert à dessiner tous les éléments visuels du jeu, allant des décors aux sprites des différents personnages et à leurs animations.
